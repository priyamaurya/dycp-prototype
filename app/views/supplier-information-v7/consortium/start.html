Create a consortium

Two or more suppliers that want to submit a joint bid can create a consortium. You can add as many suppliers as you need to.

As the organisation creating the consortium, you'll need to enter:

  • a consortium name
  • an address
  • an email address
  • your most recent share code
  • the most recent share codes for all the other suppliers

Any supplier you want to add to the consortium must have completed their supplier information and created a share code.

When you've added all suppliers, you'll create a share code for your combined supplier information.
